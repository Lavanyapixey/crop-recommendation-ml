# 🌾 Crop Recommendation System using Machine Learning

## 📌 Overview
This project is a Machine Learning-based Crop Recommendation System that predicts the most suitable crop based on soil and weather conditions. It is useful for helping farmers make better agricultural decisions using data-driven insights.

## 📊 Dataset
The dataset contains 2200 samples with the following features:
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- Soil pH
- Rainfall

The target variable is the crop type (rice, wheat, maize, etc.).

## 🤖 Model Used
- Algorithm: Decision Tree Classifier
- Type: Supervised Machine Learning

## ⚙️ Workflow
1. Data loading and preprocessing
2. Splitting dataset into training and testing sets
3. Model training using Decision Tree
4. Prediction on test data
5. Accuracy evaluation
6. Sample prediction testing

## 📈 Result
The model achieved an accuracy of approximately **99%**, showing strong performance in predicting suitable crops based on environmental conditions.

## 🛠 Tools Used
- Python
- Google Colab
- Pandas
- Scikit-learn

## 🌱 Outcome
This system successfully recommends suitable crops based on soil and weather parameters, helping improve agricultural productivity.
